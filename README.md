## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ and npm
- The processed data file: `courses_processed.json` (generated by ETL script)

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The application will be available at `http://localhost:3000`
